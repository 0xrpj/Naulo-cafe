Instructions to run the program

1.	Install ‘node js’

2.	Install ‘react js’ and ‘json server’

3.	Install the packages of ‘react-slick’, ‘axios’, ‘react-router’, ‘react-router-dom’ and
 ‘jquery’. 

4.	Use npm start in command prompt to run the program

5.	Use ‘ json-server  -p  5555  --watch for-all.json’ in cmd to run the json file located in the folder ‘public/json’.

6.	All necessary files are stored in the the 'public' folder.
